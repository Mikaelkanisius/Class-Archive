HTML

HTML merupakan bahasa dasar untuk membuat web. HTML atau HyperText Markup Language merupakan sebuah bahasa markah untuk membuat halaman web Jadi, HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web.
